# Java-EjercicioMatrices

In this project, two for loops will be used so that it can allow the teacher to load the student's grades in the three positions of a table (3 columns of each row), and that the averages are calculated in the last column. Once the calculations are done, you want to show the 3 grades of each student and the corresponding average by going through the matrix
